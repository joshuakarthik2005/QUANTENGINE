QuantEngine – Statistical Diagnostics
Privacy Policy

Last updated: December 7, 2025

Overview

QuantEngine – Statistical Diagnostics ("the extension") is a browser extension that provides local statistical analysis tools in a side panel. Users can paste or upload numeric data (for example, returns or PnL values) and view computed statistics and visualizations.

Data Collection

This extension does not collect, store, transmit, or share any personal data, browsing data, or website content.

  • No personal identifiers (name, email, address, etc.) are collected.
  • No browsing history or page content is read or stored.
  • No analytics or tracking services are used.
  • No data is sent to external servers; all computation happens locally in your browser.

User-Provided Data

Any numeric data that you paste or upload into the extension is processed entirely on your device. It is used only to compute the displayed statistics and plots and is not transmitted elsewhere. Your data never leaves your browser.

Permissions

The extension uses the sidePanel permission only to display its user interface in Chrome's side panel. It does not read or modify the content of any web page.

Third-Party Services

This extension does not use any third-party services, APIs, or analytics platforms. All statistical computations are performed locally using built-in JavaScript algorithms.
